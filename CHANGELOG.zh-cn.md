# Change Log

All notable changes to the "ra2-ini-intellisense" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [Unreleased]

- 暂无未发布变更。

## [0.0.4] - 2026-06-09

- 新增 `.llf` 语言支持，提供标签、值、注释和多行续行内容的专用语法高亮。
- 新增活动栏中的 INI 与 MIX 专属浏览视图，并采用懒加载方式提升大工作区与大档案下的响应速度。
- 新增虚拟 `ra2mix:` 文件系统，可直接将 `.mix` 档案作为工作区打开，也可在新窗口中打开。
- 新增 MIX 档案管理能力，支持导入、导出、重命名、删除、新建目录等常用操作。
- 新增资源预览器，默认支持打开 `pcx`、`pal`、`map`、`mpr`、`yrm`、`shp`、`vxl` 和 `hva` 文件。
- 新增 CSF 大纲视图，可直接在侧边栏浏览字符串表内容。
- 新增资源对比工作流，可递归比较 MIX 档案与文件夹，并提供结果汇总、筛选、文本差异和直接打开操作。
- 重构资源浏览布局，聚焦于 INI 文件、MIX 文件与 CSF 内容，而不是旧的一体化大纲视图。
- 改善 SHP 预览的调色板选择与缺失调色板时的回退处理。
- 提升 VXL/HVA 预览能力，支持切片模式与透视/游戏视图两种查看方式。
- 将体素资源的默认打开模式切换为更接近游戏实际表现的游戏视图。
- 优化体素交互体验，补充旋转、平移、缩放、部件切换、相机重置、覆盖式控件和更适合小编辑器尺寸的布局。

## [0.0.3] - 2025-11-22

- 新增本地化欢迎页与设置向导，帮助首次使用时完成 Schema 与项目路径配置。
- 新增自动注册节辅助能力，方便维护各类注册表内容。
- 新增按错误码单独设置诊断等级的能力，可对每条规则分别降级、升级或禁用。
- 切换到更明确的 `ra2-ini` 语言标识、默认语义着色配置和更合适的 scope 命名，提升高亮稳定性。
- 用官方推荐的语义高亮路径替代旧的自定义主题着色方案，改善编辑器兼容性。
- 将问题检测从全文刷新优化为增量解析，并优先处理当前可见区域，提高大文件下的响应速度。
- 将索引规则从简单的 include 列表重构为 Rules、Art、Sound、UI、Theme、AI 等文件分类匹配方式。

## [0.0.2] - 2025-11-13

- 新增基于 Schema 的 RA2 INI 智能感知，包括键补全、类型悬停和跨文件跳转到定义。
- 新增 `INIValidator.exe` 集成，支持配置可执行文件路径与校验根目录。
- 新增活动栏中的 INI 项目浏览器，用于浏览已索引的节内容。
- 新增节上方的 CodeLens 引用计数，以及继承覆盖指示器。
- 新增可配置的内置诊断，覆盖空白、注释风格与值校验相关行为。
- 新增可自定义的语法配色、Schema 路径命令、调试辅助命令与校验器管理入口。
- 新增面向 `.ini` 文件的 RA2 专用语言注册。
- 扩展了索引文件、校验目标与编辑器行为相关的配置项。
- 改善常用 INI 编辑流程的命令入口与上下文菜单集成。

## [0.0.1] - 2024-12-28

### 新增

- 首次发布到插件商店。
- 新增 INI 节、键、值与注释的语法高亮，并支持较复杂的 `[]:[]` 节语法。
- 新增节与值说明的悬停提示。
- 新增基础诊断，覆盖常见的空格与注释格式问题。
- 新增按节折叠代码块的能力。
- 新增与 INI 键值相关的跳转到定义能力。
